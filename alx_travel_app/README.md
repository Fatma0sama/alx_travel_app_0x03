# ALX Travel App 0x03

This project is a duplication of `alx_travel_app_0x02` with **Celery + RabbitMQ** background task support for sending booking confirmation emails.

## Features

- Django REST Framework backend
- Booking system
- Background email notifications using Celery
- RabbitMQ as message broker
